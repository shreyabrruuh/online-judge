## Tech Stack

**Client:** Vite+React.js, TailwindCSS

**Server:** Node, Express

**Database:** MongoDB

## To Run this project

To run frontend of this project run

```bash
  npm run dev
```

To run backend and compiler of this project run

```bash
  node index.js
```

## Features

- Problems on various DSA topics
- Time bound practice
- Test any code on compiler
- Cross platform
